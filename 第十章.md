配置Telnet
	
	Telnet简介
		远程控制web服务器的方法
	
	Telnet工作过程
		服务器采用C/S工作模式
		
		本地客户端程序向远程服务器程序发起连接请求,需要提供远程服务器的ip和域名
		远程服务器响应本地计算机的telnet连接请求并等待接收命令
		连接成功后,客户端把本地计算机模拟成远程服务器的终端,输入任何命令和数据直接传送到远程服务器
		服务器执行或处理接到的命令和数据,并把结果返回给本地计算机的终端
		
	安装和使用Telnet
		sudo yum -y install telnet
